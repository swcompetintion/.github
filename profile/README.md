# 수정사항 반영하는 법
1. 먼저 frontend 레포지토리에서 변경사항이 있을 경우(프런트 레포지토리 변경사항 없으면 안 해도 됨) git push (main 레포지토리로)
2. 이후 backend 레포지토리에서 변경사항이 있을 경우(백엔드 레포지토리 변경사항 없으면 안 해도 됨) git push (main 레포지토리로)
